# atlas-data

Data repo for the Dispensary Atlas all-jurisdictions product-search shards.

**The product-search site is on hold.** The all-jurisdictions shards (`products/*.json`,
`stores.json`) that this repo served to the `/search-all/` shell have been removed. The
generator tooling still lives in the private monorepo (`scripts/export_products.py`), so the
shards can be regenerated and re-published if the search site is resumed.
